# Design Mockups as Reference Only

Selected AI-generated UI mockups can be kept as design references under `docs/design/`,
but they should not become production assets. They are used to communicate density,
hierarchy, and visual direction before implementing real React components and CSS.

**Consequences**

- Only useful final mockups and concise notes should be retained.
- Production UI remains component-based and maintainable.
- Implementation agents can use design references without treating generated images as
  shippable UI assets.
